# PMS-Tools

用于酒店 PMS 管理软件辅助效率工具，生成账单，计算价格等功能。

### 安装

支持 Chromium 内核浏览器，建议使用 Microsoft Edge 浏览器安装使用，与 Google Chrome 浏览器兼容，不影响使用。
